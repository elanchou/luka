# App Review Notes

No account, payment flow, or external login is required for review.

Review steps:
1. Launch the app.
2. Choose "Create Master Password" on first launch.
3. Create a local vault password and enter the main dashboard.
4. Add a seed phrase, private key, or secure note locally.
5. Optional: test export and iCloud backup flows if desired.

The app stores vault content locally in encrypted form. Optional iCloud backup uses the review device's own Apple account and is not required to complete review.
