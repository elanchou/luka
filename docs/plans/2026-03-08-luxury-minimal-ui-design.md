# Luxury Minimal UI Design

**Product Goal:** Refresh the app into a modern, premium, understated vault experience with a luxury-minimal visual system while preserving existing flows and security behavior.

## Visual Direction

- Palette: obsidian black, graphite, warm silver, muted champagne accent.
- Mood: quiet, premium, disciplined, private.
- Contrast: crisp type on dark surfaces with restrained highlights instead of bright neon accents.
- Motion: subtle fades and elevation shifts only; no aggressive glow-heavy effects.

## Core Principles

- Use whitespace and proportion as the primary luxury signal.
- Reduce decorative noise: remove heavy gradients, grids, and loud icon treatments.
- Make every high-frequency screen feel like the same product through shared cards, form controls, spacing, and typography.
- Keep security cues understated and elegant rather than alarmist.

## Component System

- Backgrounds become soft layered dark surfaces with low-contrast radial light and premium depth.
- Buttons become flatter, denser, and more refined with tighter typography and softer highlights.
- Input fields move from underline-only to structured panel fields with clear focus states.
- Brand and headers become calmer, with more deliberate letter spacing and lighter chrome.
- Bottom navigation becomes thinner, quieter, and more integrated into the overall surface system.

## Page Targets

- Splash/onboarding: premium brand-first entry with restrained copy and atmosphere.
- Master password screens: centered, card-driven authentication surfaces with strong spacing and minimal distractions.
- Dashboard: executive summary feel with a quieter search area, premium list cards, and lighter navigation chrome.
- Secret detail: content-forward layout with elegant reveal behavior and cleaner metadata hierarchy.
- Settings: grouped system panels with improved rhythm and more premium switches/tiles.

## Non-Goals

- No route changes.
- No provider/state refactors for visual work.
- No major interaction redesign for multi-step add-secret flows in this pass.
- No attempt to redesign every low-frequency or emergency screen.
